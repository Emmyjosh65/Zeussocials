# ZeusSocials - Social Media Marketing Platform

## Overview
ZeusSocials is a professional social media marketing platform that allows users to purchase promotion services, with a **FREE TIER** for likes, followers, and views.

## Features
- 🎯 Multiple social media services (TikTok, Instagram, Facebook, YouTube, Telegram, Twitter/X)
- 💰 Paid services with real-time price calculator
- 🎁 **FREE TIER** - Free likes, followers, and views
- 👤 User authentication & dashboard
- 📊 Order tracking & management
- 💳 Payment integration (Paystack + Manual OPay)
- 👨‍💼 Admin dashboard for order & payment management
- 🌙 Dark mode with blue/purple theme
- 📱 Fully responsive design

## Tech Stack
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Firebase (Auth, Firestore, Storage)
- **Payment**: Paystack API + Manual OPay
- **Hosting**: GitHub + Render

## Setup Instructions
1. Clone the repository
2. Update Firebase credentials in `js/firebase-config.js`
3. Deploy to Render or GitHub Pages
4. Configure Paystack API keys
